# 🌐 Bootstrap 5 UI Project

## Live Demo

https://nikitha26reddy.github.io/bootstrap-ui-project/

## Project Overview

This project is a responsive multi-page website built using Bootstrap 5.
The goal was to explore Bootstrap components and combine them creatively to design clean and modern web pages.

## Pages Included

* 🏠 Home Page
* ℹ️ About Page
* 📞 Contact Page

## Features

* Responsive navigation bar
* Hero section with call-to-action
* Feature cards with icons
* Carousel for visual interaction
* Contact form with validation
* Clean and modern UI design

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* Bootstrap Icons

## Responsiveness

The website is fully responsive and works across:

* Mobile devices
* Tablets
* Desktop screens

This was achieved using Bootstrap’s grid system and utility classes.

## Learning Outcomes

* Understood Bootstrap components and layout system
* Learned how to design responsive web pages
* Improved UI/UX design sense
* Gained experience in combining multiple components

## Challenges Faced

* Aligning components properly using grid
* Maintaining consistent spacing
* Making design responsive across devices

## Solutions

* Used Bootstrap grid system effectively
* Applied spacing utilities like margin and padding
* Tested responsiveness using browser tools

## Tools & Resources

* Bootstrap Documentation
* Bootstrap Examples
* ChatGPT (for guidance and learning support)

I explored Bootstrap examples and documentation to understand components.
Instead of copying templates, I combined different components like navbar, cards, carousel, and forms to build a clean and responsive layout.
